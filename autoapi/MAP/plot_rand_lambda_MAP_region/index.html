MAP.plot_rand_lambda_MAP_region.plot_lambdas_3D(file_path, x_value, y_value, z_value, output_prefix)

Function to plot optimized_lambda and Dirichlet_lambda_ref in a 3D plot with semi-transparent dots, overlaying one over the other.

Parameters:

  • file_path – Path to the CSV file containing lambda values

  • x_value – The name of the value for the x-axis (e.g., ‘gpt2-helpful’)

  • y_value – The name of the value for the y-axis (e.g., ‘gpt2-harmless’)

  • z_value – The name of the value for the z-axis

  • output_prefix – Path to save the output PDF

MAP.plot_rand_lambda_MAP_region.plot_lambdas_2D_subplots(file_paths, x_value, y_value, output_prefix)

Function to create several subplots, one for each file, in a 2D plot with semi-transparent dots. The Dirichlet_lambda_ref is plotted file-specific for each subplot.

Parameters:

  • file_paths – List of paths to the CSV files containing lambda values

  • x_value – The name of the value for the x-axis (e.g., ‘gpt2-helpful’)

  • y_value – The name of the value for the y-axis (e.g., ‘gpt2-harmless’)

  • output_prefix – Path to save the output PDF
